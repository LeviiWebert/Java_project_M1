Compilation :

Pour compiler le projet à partir du terminal, se diriger vers le dossier src compris dans 
    le projet puis executer les commandes suivantes: 

cd : C:\chemin\vers\le\repertoire

java -jar BahBrahimi.jar


Pour compiler le projet à partir d'un IDE, il faut exécuter le fichier BahBrahimi.java




Utilisation de MySQL pour la Base de données "project_java_db" qui est jointe dans le dossier de transmission.
